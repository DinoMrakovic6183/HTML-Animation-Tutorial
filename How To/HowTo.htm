How To Make Animations With CSS

First, you have to have a class for the animation to be under, preferably a div class.

Example:

< div class = "(insert class name)" >< /div >

After that, you have to open the class in your style section or in your CSS file

Example:

(class name) {
animation-...
}

Once you made your class, you can start on the animation part. (you can use width and height to make a square as well)

Example:

(class name) {
position: relative;
animation-duration:#s; this is the length of the animation
animation-delay:#s (or 0); this is the delay of the start of the animation
animation-iteration-count:# (or "infinite"); this is how many times the animation repeats
animation-name:(insert name of animation); this is the name of the animation used in the next example
}

After this is finished, you need to create an @keyframes line with your animation name

Example:

@keyframes (name of animation) {
...
}

Now that you have your @keyframe line opened, there are two ways of making the animation (a to and from, and a % movement)

The first way is to put the code

"from {(direction):#px (typically 0);}" and "to {(direction):#px;}"

The second way is to put the code

"0%{left:0px; top:0px;}
25%{left:#px; top:0px;}
50%{left:#px; top:#px;}
75%{left:0px; top:#px;}
100%{left:0px; top:0px;}"







My HTML:

< div class = "hi" > < /div >

My CSS:

div.hi {
background-color:red;
width:100px;
height:100px;
position: relative;
animation-name: example;
animation-duration: 5s;
animation-iteration-count: infinite;
}


@keyframes example {
0% {left:0px; top:0px;}
25% {left:500px; top:0px;}
50% {left:500px; top:100px;}
75% {left:0px; top:100px;}
100% {left:0px; top:0px;}
}
